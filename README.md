## 📖 專案概述 (Project Overview)

本專案為台灣主婦聯盟生活消費合作社（新竹分社）專屬打造的\*\*「分社層級委員會通用版入口網」\*\*。

為解決過去各委員會（生活材、教育、編輯、福祉）開會前，委員們總是在 LINE 群組或記事本中翻找「會議連結、議程表、提案表單」的行政痛點，本系統將所有開會必備資訊集中於單一雲端介面。透過輕量化的密碼驗證與動態資料抓取技術，提供委員們一個安全、直覺且永遠保持在最新狀態的議事大廳。

## ✨ 核心特色 (Core Features)

1.  **無頭式資料庫 (Headless Database)**：以 Google Sheets 作為後台資料庫（發佈為 CSV），前端透過 JavaScript `fetch` 即時讀取最後一筆資料，讓行政夥伴只需在試算表填入新連結，網頁端即刻同步，無需更改任何程式碼。
2.  **動態單位切換**：整合四大委員會（🛒 生活材、🎓 教育、🍀 編輯、❤️ 福祉），在下拉選單切換時，下方的功能按鈕會自動且平滑地重新綁定對應單位的專屬連結。
3.  **輕量級門禁系統**：內建基礎的通行碼驗證機制，防止非內部人員誤入會議或隨意瀏覽內部文件，兼顧便利性與基礎資訊安全。
4.  **四大直覺化模組**：
      * **📅 今日議程**：一鍵開啟當次會議的共筆或 PDF 議程。
      * **📝 提交提案**：統一導流至跨委員會通用的 Google Forms 提案表單。
      * **🎥 線上會議**：直達 Google Meet 會議室，免去群組翻找困擾。
      * **📚 歷次紀錄**：快速跳轉至該委員會的歷史會議紀錄歸檔區。
5.  **跨裝置友善 (RWD)**：採用 Bootstrap 5 框架與大按鈕（Big UI）設計，無論委員使用電腦或手機瀏覽器，皆能獲得極佳的點擊與閱讀體驗。

## 🛠️ 技術架構 (Tech Stack)

  * **前端介面**: HTML5, CSS3 (自訂變數與漸變動畫), Vanilla JavaScript (ES6+)
  * **UI 框架**: Bootstrap 5.3.0
  * **資料庫與後台**: Google Sheets (Publish to Web as CSV) & Google Forms
  * **非同步處理**: `async/await`, Fetch API
  * **部署環境**: 靜態網頁代管服務 (如 GitHub Pages)
